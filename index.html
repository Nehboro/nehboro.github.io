What is Nehboro

A cleaner, calmer interface for serious threat intelligence

Nehboro blends feed-based blocking with deep heuristic page analysis across ClickFix, phishing, and malware-delivery techniques. Everything stays transparent, local, and easy to inspect.

🛡️

Feed-based blocking

Domains, URLs, IPs, CIDRs, wildcards, and suspicious ports are auto-refreshed every six hours directly from GitHub Pages.

🔍

Heuristic detection

Catches ClickFix chains, fake Cloudflare flows, clipboard hijacks, PowerShell delivery, macOS shell abuse, and more — including brand new domains.

🚩

Community reporting

One click sends structured evidence through ntfy.sh. Score, detections, and context are bundled so confirmed IOCs can quickly land in public feeds.

🌙

Transparent by design

Every IOC is visible, custom feeds are supported, trusted domains can be exempted, and every block can be bypassed deliberately.

How it works

Up and running in minutes

Load once, protect continuously. The interface stays simple while the detection logic stays opinionated.

1

Install the extension

Download it from GitHub Releases. Set your secret topic in background.js, then load it from chrome://extensions with Load unpacked.

2

Feeds load automatically

All four feeds are fetched from nehboro.github.io/feeds/ and converted into browser blocking rules on install, then refreshed every six hours.

3

Every page is analyzed

Content scripts score each page across ClickFix, phishing, and malware-delivery patterns. Scores over your thresholds trigger warnings or blocking with a deliberate bypass option.

4

Report threats back to the community

Use the report action to send score, detections, and evidence to your private ntfy.sh topic. Confirmed IOCs can then be added to the shared feeds.

# 1. Set your secret ntfy topic in background.js
const NTFY_TOPIC = 'nehboro-reports-YOUR_SECRET';
 
# 2. Load unpacked in Chrome
chrome://extensions → Developer mode → Load unpacked → nehboro_v2/
 
# 3. Read your private threat reports
https://ntfy.sh/nehboro-reports-YOUR_SECRET
